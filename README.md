# nuxt-example

Basic Nuxt example

## Set up for local development

Pre-requisites:
- [NVM](https://github.com/nvm-sh/nvm)

Clone repository:

```shell
git clone git@github.com:angelxehg/nuxt-example.git
cd nuxt-example
```

Install Node 20 with NVM:

```shell
nvm install 20.13.1
nvm use 20.13.1
```

Install dependencies:

```shell
npm install
```

Start application:

```shell
npm run dev
```
